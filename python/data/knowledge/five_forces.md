# 造浪五力能力框架

## 好奇心（Curiosity）
对世界保持开放和探索的态度，主动提出问题，愿意尝试不熟悉的事物。

**观察指标**：
- 提问的频率和质量（是否追问"为什么"和"如果...会怎样"）
- 面对陌生事物时的第一反应（回避 vs 靠近）
- 自主探索行为的持续性（不需要外部奖励也能深入）

**典型行为**：
- 在森林徒步中主动发现并提出关于动植物的问题
- 对工具或材料表现出"我想试试"的态度
- 在自由活动时间选择探索而非等待指令

**发展阶段**：
- 萌芽期（0-30）：在熟悉的领域内探索，需要引导才能迈出舒适区
- 成长期（30-55）：主动尝试新事物，遇到感兴趣的话题会深入追问
- 发展期（55-75）：保持持续的好奇心，能从多个角度提出问题
- 绽放期（75-100）：自带驱动力地探索未知，问题有深度和系统性

## 创造力（Creativity）
能够产生独特的想法，用多种方式解决问题，通过不同媒介表达自我。

**观察指标**：
- 想法的原创性（是模仿还是创新）
- 解决问题时的方案多样性
- 艺术表达的独特性和完成度

**典型行为**：
- 手工制作中展现了超出指导范围的独特设计
- 面对开放式任务时提出多个不同的解决方案
- 在戏剧/故事创作中有自己的风格

**发展阶段**：
- 萌芽期（0-30）：倾向于模仿和跟随，在给定框架内完成任务
- 成长期（30-55）：在给定主题下能产生独特想法，有自己的表达偏好
- 发展期（55-75）：能主动创造，想法有独特性，作品有个人风格
- 绽放期（75-100）：创意新颖有深度，能在不同媒介间自如切换

## 协作力（Collaboration）
在团队中有效合作，能够倾听他人、表达自己、解决冲突。

**观察指标**：
- 在团队中的角色（跟随者 / 贡献者 / 协调者）
- 倾听和回应他人的质量
- 面对团队冲突时的行为

**典型行为**：
- 主动帮助团队中遇到困难的成员
- 在讨论中既能表达自己观点也能认真听别人说
- 团队意见不一致时能提出折中方案

**发展阶段**：
- 萌芽期（0-30）：倾向于独立完成任务，需要鼓励才能参与团队
- 成长期（30-55）：能配合团队完成任务，在分工中尽责
- 发展期（55-75）：主动协调团队，能发现并支持他人需求
- 绽放期（75-100）：自然的团队凝聚者，能激发团队成员的潜力

## 抗逆力（Resilience）
面对挑战和失败时保持积极，能从挫折中恢复并继续尝试。

**观察指标**：
- 面对困难时的第一反应（放弃 / 求助 / 再试）
- 从挫折中恢复的时间和方式
- 对失败的解释方式（"我不行" vs "这个方法不行"）

**典型行为**：
- 挑战失败后主动调整策略再次尝试
- 在野外或陌生环境中适应良好
- 被批评后能接受反馈并改进而非抵触

**发展阶段**：
- 萌芽期（0-30）：遇到挫折容易气馁，需要外部鼓励才能继续
- 成长期（30-55）：遇到困难会尝试几次，懂得寻求帮助
- 发展期（55-75）：能自我调节情绪，挫折后恢复较快
- 绽放期（75-100）：视挑战为成长机会，失败后快速调整再尝试

## 表达力（Communication）
能够清晰地表达自己的想法和感受，善于讲故事，在公众场合敢于发声。

**观察指标**：
- 语言表达的清晰度和逻辑性
- 叙述的生动性和感染力
- 在公众场合的自信程度

**典型行为**：
- 分享环节中能清晰地说出自己的经历和感受
- 讲故事时有细节、有情感、有结构
- 在团队汇报中能自信地表达

**发展阶段**：
- 萌芽期（0-30）：在熟悉的小范围内表达，公众场合较为内向
- 成长期（30-55）：能在小组中清晰表达，分享环节可以发言
- 发展期（55-75）：表达逻辑清晰、有感染力，敢于在众人面前发言
- 绽放期（75-100）：表达生动有趣有深度，能自信地在任何场合演讲
